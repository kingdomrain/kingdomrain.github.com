HTML 5 Web 存储

HTML5 提供了两种在客户端存储数据的新方法：

localStorage - 没有时间限制的数据存储

sessionStorage - 针对一个 session 的数据存储

之前，这些都是由 cookie 完成的。但是 cookie 不适合大量数据的存储，因为它们由每个对服务器的请求来传递，这使得 cookie 速度很慢而且效率也不高。

css3特效

实例一,移入改变大小,主要代码如下

				.div{
		width: 300px;
		height: 300px;
		border-radius: 300px;
		border: #000 solid 1px;
	}
	.cricle{
		width: 200px;
		height: 200px;
		border-radius: 200px;
		background: #F0AD4E;
		margin: auto;
		margin-top: 50px;
		text-align: center;
	}
	div{
		transition: all 2s;
		perspective:10PX;
	}
	.cricle:hover{
		transform:translateZ(-100px)
	}

实例二,创造一个3D盒子,主要代码如下

				.box{
			width: 100px;
			height: 100px;
			position: relative;
			margin: 100px auto;
			transform:rotateY(40deg) rotateX(40deg);
			transform-style: preserve-3d;
		transition: all 20s;
		}
		.box div{
			width: 100px;
			height: 100px;
			position: absolute;
			top: 0;
			left: 0;
		}
		.left{
			transform:rotateY(90deg) translateZ(-50px);
			background: #F0AD4E;
		}
		.right{
			transform:rotateY(90deg) translateZ(50px);
			background: #F45F4E;
		}
		.back{
			transform:translateZ(-50px);
			background: #0f0f99;
		}
		.front{
			transform:translateZ(50px);
			background: #0f0f99;
		}
		.top{
			transform:rotateX(90deg) translateZ(50px);
			background: #65356f;
		}
		.bottom{
			transform:rotateX(90deg) translateZ(-50px);
			background: #0f0;
		}

		.box{
			
animation:myfirst 5s infinite;
}
