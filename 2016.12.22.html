AJAX

AJAX = Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）.主要用于无需加载整个网页,只需要加载部分内容的时候.

XMLHttpRequest 对象的用法如下: 

	var xhl=new XMLHttpRequest();
	
	var variable=new ActiveXObject("Microsoft.XMLHTTP");  //ie5/6的写法,如果要处理兼容的话需要如下写法
	
	var xmlhttp;
	if (window.XMLHttpRequest)
    {
    xmlhttp=new XMLHttpRequest();
    }
	else
    {
    xmlhttp=new ActiveXObject("Microsoft.XMLHTTP");
    }

AJAX - 向服务器发送请求,有以下两种: 

	xhl.open("GET","DATA.json");  //open(method,url,async) ture代表异步 false代表同步
	
	xhl.send(null);   //将请求发送到服务器

onreadystatechange 事件

每当 readyState 改变时，就会触发 onreadystatechange 事件。readyState和status的值主要一般有以下几种 

	readyState 存有 XMLHttpRequest 的状态。
	从 0 到 4 发生变化。

0: 请求未初始化 
1: 服务器连接已建立 
2: 请求已接收 
3: 请求处理中 
4: 请求已完成，且响应已就绪 
 
status 的两种状态.
200: "OK"

404: 未找到页面

一般书写的时候会加以下判断,以确保不会出错

	if(xhl.readyState==4){ 
				if(xhl.status==200){
					
				}
			}
