for循环的复习

for循环基本语法:

			for(申明变量;表达式或值;表达式或值(后运算)){
				执行代码;
			}

实例

	 var box=document.getElementById("box");
	 var str=new String;
	 var k=3;
	 for(var i = 0; i < 5; i ++ ){
	 	for(var j = 0; j < 5; j ++){
	 		if(5*i+j-k<0){
	 		str+="<div></div>"
	 		}else{
	 		str+="<div>"+(5*i+j-k+1)+"</div>"
	 		}
	 	}
	 }
	 box.innerHTML=str;

运算符的复习

常用运算符

		
		负值 - 逻辑非 ! 按位取反 ~ 赋值 = 删除 delete
		递增 ++ 小于 < 按位左移 << 运算赋值 oP= typeof 运算符 typeof
		递减 -- 大于 > 按位右移 >> void 运算符 void
		乘法 * 小于等于 <= 无符号右移 >>>
		除法 / 大于等于 >= 按位与 &
		取模运算 % 等于 == 按位异或 ^
		加法 + 不等于 != 按位或 |
		减法 - 逻辑与 &&
		逻辑或 ||
		条件（三元运算符） ?:
		逗号 ,
		恒等 ===
		不恒等 !==

注意+=与=+的差别,一个是先加后赋值,一个是先赋值后加

函数的复习

函数的三种声明方法

1.匿名声明

			function(){
		    alert(123)
			}

2.表达式声明

			var fun=function(){
		    alert(123)
			}

3.直接声明

			function fun(){
		    alert(123)
			}

直接申明的函数可以在声明函数的上方调用,如:

			name();
			function name(){
				alert(1)
			}

函数也可以用同样的名字覆盖自己,不过不同的声明方式会有不同的优先度,如:以下情况下会弹出3

			function name(){
				alert(1)
			}
			
			var name=function(){
				alert(3)
			}
			function name(){
				alert(2)
			}
			name();
			}

以下情况下会弹出2

			function name(){
				alert(1)
			}
			function name(){
				alert(2)
			}
			
			name();
			}
