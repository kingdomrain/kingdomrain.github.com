html5标签

html5标签的特性主要有:

语义化：能够让你更恰当地描述你的内容是什么

连通性：能够让你和服务器之间通过创新的新技术方法进行通信

离线存储：能够让网页在客户端本地存储数据以及更高效地离线运行

多媒体：使 video 和 audio 播放音视频

绘图/动画效果：提供了一个更加分化范围的呈现选择

性能集成：提供了非常显著的性能优化和更有效的计算机硬件使用

设备访问 ：能够处理各种输入和输出设备

兼容性：百度CND：http://apps.bdimg.com/libs/html5shiv/3.7/html5shiv.min.js

html5标签的练习,主要代码如下

			<header><header></header>

			<article><article></article>

		<aside><aside>
			<div class="widget"><div class="widget"></div>
			<div class="widget"><div class="widget"></div>
		</aside>
		<footer><footer></footer>

sqlLita数据库

SQLite，是一款轻型的数据库，是遵守ACID的关系型数据库管理系统，它包含在一个相对小的C库中。它是D.RichardHipp建立的公有领域项目。它的设计目标是嵌入式的，而且目前已经在很多嵌入式产品中使用了它，它占用资源非常的低，在嵌入式设备中，可能只需要几百K的内存就够了。它能够支持Windows/Linux/Unix等等主流的操作系统，同时能够跟很多程序语言相结合，比如 Tcl、C#、PHP、Java等，还有ODBC接口，同样比起Mysql、PostgreSQL这两款开源的世界著名数据库管理系统来讲，它的处理速度比他们都快。SQLite第一个Alpha版本诞生于2000年5月。 至2015年已经有15个年头，SQLite也迎来了一个版本 SQLite 3已经发布。

使用数据库的主要四个功能,增删改查,主要代码如下:

			
			INSERT INTO TABLE_NAME (column1, column2, column3,...columnN)] //添加 
			VALUES (value1, value2, value3,...valueN);
			
			SELECT column1, column2, columnN FROM table_name;  //查询
			
			UPDATE table_name   //修改
			SET column1 = value1, column2 = value2...., columnN = valueN
			WHERE [condition];
			
			DELETE FROM table_name  //删除
			WHERE [condition];
